# amplify – TUJ UI/UX Hackathon 2026

## Overview
amplify is a redesigned live music discovery platform created during the **Temple University Japan Spring 2026 UI/UX Hackathon**.

The goal of the project was to transform a provided static HTML page into a modern, user-friendly experience using **CSS and JavaScript enhancements** while keeping the original HTML structure unchanged.

Our redesign focuses on improving the **user experience, visual hierarchy, and engagement** for users searching for live music events and underground artists.

---

## Features

- **Live Music Discovery Interface**
  - Users can explore upcoming concerts and events.

- **Modern UI/UX Design**
  - Clean layout with improved visual hierarchy.
  - Smooth animations and transitions.

- **Interactive Elements**
  - Dynamic sections powered by JavaScript.
  - Responsive navigation.

- **Intro Video Animation**
  - Custom loading screen with a video intro that fades into the main page.

- **Responsive Design**
  - Optimized for multiple screen sizes.

## How to Run the Project

### Option 1: Run with Live Server (Recommended)

1. Download or clone the repository: 
git clone https://github.com/SydneyBasulto/tuj-ui-hackathon-2026

or download the ZIP from GitHub and extract it.

2. Open the project folder in **Visual Studio Code**.

3. Install the **Live Server** extension (if not already installed):
   - Go to Extensions
   - Search for **Live Server** by Ritwick Dey
   - Install the extension

4. Right-click on `index.html` in the file explorer.

5. Select:


Open with Live Server


6. The project will open in your browser at a local address
